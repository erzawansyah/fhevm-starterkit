# Automation Scripts

Dokumen ini menjelaskan berbagai skrip otomatisasi yang tersedia untuk digunakan dalam proyek ini. Skrip-skrip ini dirancang untuk memudahkan tugas-tugas umum dan mempercepat alur kerja pengembangan.

## Template Initialization Script

Skrip ini menginisialisasi template dasar ke dalam proyek Anda dengan meng-clone template dari repository yang telah ditentukan di `starterkit.config.json` ke dalam folder `./base`. Proses ini hanya perlu dilakukan sekali.

```bash
npm run template:init
```
