# Overview

This documentation provides comprehensive guides for starter examples in the fhevm-starterkit project.

## What is fhevm-starterkit?

The fhevm-starterkit is a collection of starter templates and examples demonstrating how to build decentralized applications using Fully Homomorphic Encryption (FHE) on the Zama blockchain.

## Purpose

Each example in this project serves as a practical starting point for developers who want to:

- Build privacy-preserving smart contracts
- Implement confidential computations on-chain
- Understand FHE integration in blockchain applications
- Learn best practices for encrypted data handling

## Structure

The documentation covers various starter examples, each showcasing different use cases and implementation patterns for FHE-enabled smart contracts.

## Getting Started

Browse through the individual example documentation to explore specific implementations and learn how to integrate FHE capabilities into your blockchain applications.
